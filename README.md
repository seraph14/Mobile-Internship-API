DEL → Delete user by Id

POST → Add article

GET → Get All article

PATCH → Update article by Id

DEL → Delete article by Id

POST → Add comment

GET → Get comment by Id

GET → Get All comments

GET → Get specific comment

PATCH → Update comment by Id

DEL → Delete comment by Id
